## Description
Api de uma aplicação ToDo List
Trabalho de Programação III - Sistemas de Informação Uniavan

## RUN NPM INSTALL - MIGRATIONS - START PRODUCTION
```bash
$ npm run deploy
```

# OR

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# Run Migrations
$ npx prisma migrate deploy

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
