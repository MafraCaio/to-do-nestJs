## Description
Api de uma aplicação ToDo List
Trabalho de Programação III - Sistemas de Informação Uniavan

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Run Migrations
$ npx prisma migrate deploy

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
